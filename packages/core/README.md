# @khaveeai/core

Core VRM avatar functionality and types for the KhaveeAI SDK.

## Installation

```bash
npm install @khaveeai/core
```

## Basic Usage

```typescript
import { VRMManager } from '@khaveeai/core';

const manager = new VRMManager();
// Core functionality
```

## API Reference

This package provides the core types and utilities used by all other KhaveeAI packages.

### Main Exports

- `VRMManager` - Core VRM avatar management
- Types: `ConversationProvider`, `VoiceProvider`, `AudioProvider`
- Animation utilities
